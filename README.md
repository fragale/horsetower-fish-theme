HorseTower is a clean, high-contrast Fish shell theme focused on
readability, performance and DevOps workflows.

## Features

- Two-line prompt
- Host-based accent color
- Git status (dirty, untracked, stash)
- Virtualenv support
- Vi mode indicator
- High-contrast colors for dark terminals
- Bright cyan directories and commands

## Installation (Oh My Fish)

```fish
omf install https://github.com/fragale/horsetower-fish-theme
omf theme horsetower
```

## Configuration

set -g THEME_HORSETOWER_VI_MODE yes
